# Customer Segmentation Analytics Tool

A machine learning pipeline that uses **K-Means Clustering** to automatically segment a customer database into distinct behavioral profiles based on their annual income and spending habits.

## 📊 Project Overview
This project applies unsupervised machine learning algorithms to identify core consumer groups. By grouping clusters, businesses can build data-driven, targeted marketing strategies for different consumer demographics.

### Key Features
* Automated statistical data simulation for customer behavior profiling.
* Standardized feature preprocessing using Scikit-Learn's `StandardScaler`.
* Optimized K-Means clustering configuration ($k=3$).
* Dynamic, publication-grade cluster visualization layout utilizing Seaborn and Matplotlib.
